#Drivers-Premium

# 📔What is it?
Drivers Premium is an open source application that simulates the official branch of "Road Traffic Safety Agency" of motor vehicle drivers

# ⚠️Requirements
.NET Core 3.0 => https://dotnet.microsoft.com/download/dotnet-core/3.0
